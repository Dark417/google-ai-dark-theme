# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Added
- Initial release
- Dark theme for Google Search AI Mode
- Support for google.ai and google.com/search
- Adjustable text brightness slider (40% - 100%)
- Customizable link color via hue slider (0° - 360°)
- Persistent settings using Chrome storage API
- Real-time theme updates without page refresh
- Performance optimization during scroll
- Support for dropdown menus and popups
- Premium dark popup UI with toggle and sliders

### Features
- Targets AI Mode conversation UI
- Styles source cards and citations
- Dark scrollbars
- Transparent image/icon backgrounds preserved
